HTML Cheat-Sheet

Basics Tags

  • Html Tag:
    •

    This tag is used to make the browser & text editor identify that document is html document.

  • Head Tag:
    •

    Defines the head portion of html document that contains related to document.Also shows the information that isn't displayed.

  • Body Tag:
    •

    It contains all the main content of the website such as headings, paragraphs, images, tables etc.

  • Title Tag:
    •

    Puts the name of the document in the title bar.
